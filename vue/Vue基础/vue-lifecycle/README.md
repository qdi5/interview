# vue-lifecycle
### 父子嵌套组件生命周期
- 挂载阶段：    
初始化是从父到子，渲染是从子到父
- 更新阶段：    
开始更新是从父到子，执行更新是从子到父
- 销毁阶段：
开始销毁是从父到子，执行销毁是从子到父
## Project setup
```
yarn install
```
